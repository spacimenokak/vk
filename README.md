# QuizLive — Платформа для проведения квизов в реальном времени

MVP веб-приложения для создания и проведения интерактивных квизов с поддержкой WebSocket.

## Функциональность

- **Регистрация и авторизация** — роли «Участник» и «Организатор»
- **Создание квизов** — категории, время на вопрос, разные типы вопросов
- **Типы вопросов** — текстовые и с изображением; одиночный и множественный выбор
- **Real-time сессии** — подключение по коду комнаты через Socket.IO
- **Подсчёт баллов** — автоматический лидерборд по завершении
- **Личный кабинет** — история участия и проведённых квизов

## Технологии

| Слой | Технология | Обоснование |
|------|-----------|-------------|
| Frontend | React + Vite | Быстрая разработка, компонентный подход |
| Backend | Node.js + Express | Единый язык (JS) на клиенте и сервере |
| Real-time | Socket.IO | Надёжная работа WebSocket с fallback |
| БД | SQLite (better-sqlite3) | Простота развёртывания, не требует отдельного сервера |
| Auth | JWT + bcrypt | Стандартная схема аутентификации |

## Быстрый старт

```bash
# Установка зависимостей
npm run install:all

# Инициализация БД
cd server && npm run init-db && cd ..

# Запуск (сервер + клиент)
npm run dev
```

- Клиент: http://localhost:5173
- Сервер API: http://localhost:3001

## Сценарий использования

1. **Организатор** регистрируется с ролью «Организатор»
2. Создаёт квиз: добавляет вопросы, настраивает время и категорию
3. Нажимает «Запустить» — получает код комнаты (например, `ABC123`)
4. **Участники** регистрируются, вводят код на странице «Войти в квиз»
5. Организатор нажимает «Начать квиз» — вопросы показываются всем синхронно
6. Участники отвечают только на текущий вопрос
7. Организатор переключает вопросы кнопкой «Следующий вопрос»
8. По завершении — лидерборд с результатами, данные сохраняются в БД

## API

| Метод | Endpoint | Описание |
|-------|----------|----------|
| POST | `/api/auth/register` | Регистрация |
| POST | `/api/auth/login` | Авторизация |
| GET | `/api/quizzes` | Список квизов организатора |
| POST | `/api/quizzes` | Создание квиза |
| POST | `/api/sessions/start` | Запуск сессии (код комнаты) |
| GET | `/api/sessions/history` | История |

### WebSocket события

| Событие | Направление | Описание |
|---------|-------------|----------|
| `join-room` | Client → Server | Подключение к комнате |
| `start-quiz` | Organizer → Server | Начало квиза |
| `question-show` | Server → All | Показ вопроса |
| `submit-answer` | Participant → Server | Ответ участника |
| `quiz-finished` | Server → All | Лидерборд |
